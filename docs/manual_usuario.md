# Manual do Usuário - Titanium Clínica

## Introdução

Bem-vindo ao sistema Titanium Clínica, uma ferramenta para auxiliar na confirmação humanizada de consultas via WhatsApp.

## Funcionalidades Principais

### 1. Login
- Digite seu usuário e senha
- O sistema suporta diferentes perfis: admin, gestor, atendente

### 2. Gerenciamento de Pacientes
- Visualize lista de pacientes
- Prepare mensagens personalizadas
- Marque status de confirmação

### 3. Preparação de Mensagens
- Gere mensagens humanizadas automaticamente
- Revise antes do envio
- Cole no WhatsApp Web (não envia automaticamente)

### 4. Dashboard
- Visualize estatísticas diárias
- Gráficos de confirmações

## Segurança
- Nunca envia mensagens automaticamente
- Sempre revise antes de enviar
- Respeita limites anti-bloqueio
