# Política de Privacidade e LGPD - Titanium Clínica

## Compromisso com a Privacidade

A Titanium Clínica está comprometida com a proteção dos dados pessoais dos pacientes, em conformidade com a Lei Geral de Proteção de Dados (LGPD).

## Dados Coletados

- Nome completo
- Telefone
- Email (opcional)
- CPF (opcional)
- Dados de consultas

## Finalidades

- Confirmação de consultas
- Lembretes
- Comunicação sobre reagendamentos

## Direitos dos Titulares

- Acesso aos dados
- Correção de dados
- Exclusão de dados
- Revogação de consentimento

## Segurança

- Criptografia de dados
- Controle de acesso
- Logs de auditoria
- Backup regular

## Contato

Para exercer seus direitos, entre em contato conosco.
